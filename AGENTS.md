Максимальное и минимальное значение помогают понять задачи следующих этапов. При большом отличии этих показателей в различных параметрах нам будет дополнительно нужна стандартизация или нормализация - приведение данных к диапазону [0;1], где 0 - минимальное, 1 - максимальное значение.

Предобработка данных является фундаментальным этапом статистического анализа, включающим очистку, трансформацию и подготовку данных для дальнейшего исследования. Бутстреп, выравнивание групп при малых (менее 30) размерах группы или выраженной неравности групп соответственно.


Визуализация

Визуализация позволяет наглядно представить распределение данных и выявить основные закономерности. Включает построение различных типов графиков для качественного анализа информации.

Скрипичные диаграммы демонстрируют распределение данных с учетом плотности вероятности, дополняются статистическими показателями (медиана, квартили).

Роевые диаграммы отображают распределение категориальных данных с добавлением статистических метрик, таких как среднее значение и стандартное отклонение.

Диаграммы “ящик с усами” визуализируют размах данных и значимые различия между группами, отмечая статистическую значимость различий специальными маркерами.

Гистограммы показывают распределение частот значений с дополнительными статистическими характеристиками (среднее, медиана, мода).

Круговая диаграмма используется для визуализации пропорций категориальных данных в процентном соотношении.

Анализ распределения показателя включает изучение формы, центра и разброса данных для определения типа распределения.

Скаттерограмма с регрессионным анализом визуализирует взаимосвязь между двумя переменными с построением линии тренда.

Корреляционная матрица отображает взаимосвязи между множеством переменных, где незначимые связи представлены полупрозрачными элементами.

3 и более параметров мы тоже можем сравнить. Для этих целей лучше использовать решетку (grid plot) с попарным сравнением.

Тесная (более 0,9, без учета знака) корреляция как правило говорит о возможности предсказать изменения одного показателя через изменения другого. Подобные корреляции позволяют исключать один из пары связанных показателей из последующей обработки.

Методы множественных сравнений

Методы множественных сравнений позволяют выявлять статистически значимые различия между группами при множественном тестировании.

Анализ частоты встречаемости категориальных переменных оценивает распределение значений по категориям.

Критерий Краскела-Уоллиса применяется для сравнения трех и более независимых выборок при непараметрических данных.

Factorial ANOVA анализирует влияние нескольких факторов и их взаимодействия на исследуемый показатель.

Factorial ANOVA (Многофакторный ANOVA): Анализирует влияние двух или более независимых факторов на одну зависимую переменную.

Оценка качества прогнозирования

Принятие решения о том, какие переменные разделяют (то есть «дискриминируют») возникающие наборы данных (так называемые «группы»).

Используется для предсказания значения выходной переменной по значениям входных переменных (предикторов). Предикторы - числовые значения из списка выбранных для анализа числовых переменных. Выходная переменная - категориальная переменная param_group, выбранная в блоке "Выбор параметров".

Разделение набора данных на учебные и проверочные данные. При больших выборках (сотни наблюдений) можно разделить набор, выделив 90% строк на учебные данные и 10% — на проверочные. На малых выборках: 50% — учебные данные и 50% — проверочные.

Точность предсказаний можно проверить с помощью кроссвалидации - последовательного использования частей выборки как учебных и проверочных.

Линейный дискриминантный анализ используется для классификации объектов по группам на основе множества признаков.

Логистическая регрессия применяется для прогнозирования категориальных зависимых переменных.

Регрессионный анализ изучает взаимосвязь между зависимой и независимыми переменными.

Деревья решений представляют собой древовидную модель принятия решений на основе признаков.

XGBoost — это один из вариантов реализации деревьев решений (Decision Tree). Он использует обратный спуск Gradient Boosting — подход, при котором новые модели обучаются прогнозировать ошибки предыдущих моделей. За счет этого метод менее подвержен ошибкам переобучения, когда при увеличении количества итераций точность прогноза не меняется / уменьшается. Метод XGBoost эффективен для обработки как числовых, так и категориальных данных.

Метод опорных векторов (SVM) использует различные ядра (RBF, polynomial) для классификации данных.

В прогностической матрице по оси Y отмечается, к какой группе (из param_group) относится наблюдение. По оси X - к какой группе его отнес классификатор. Правильная классификация будет по нисходящей диагонали.

Выбор признаков
Далее мы найдем параметры, наиболее сильно влияющие на изменчивость выборки.

Один из самых популярных и надёжных методов разработки прогностических моделей заключается в использовании алгоритма «случайный лес» (Random Forest, RF).
1) Одномерный выбор признаков и алгоритм «случайный лес» - улучшенная версия

Попробуем уменьшить количество используемых для классификации переменных. Задача - снизить сложность модели, увидеть какие связи являются самыми значимыми. В ряде случаев это позволяет увеличить точность прогноза.
2) рекурсивное устранение признаков (rfe) и лес решений

При использовании данного метода, в обучающих данных не должно быть коррелированных признаков. Случайный лес решений выбирается случайным образом на каждой итерации, поэтому последовательность списка важности признаков может меняться.

Преобразование признаков для уменьшения их количества
С использованием методики снижения размерности пространства признаков можно представить исходный набор данных через меньшее количество переменных и при этом снизить объём вычислительных ресурсов, необходимых для обеспечения работы модели. Используя метод главных компонент (PCA), можно изучить кумулятивную выборочную дисперсию этих признаков для того чтобы понять то, какие признаки объясняют большую часть дисперсии в данных. Но прежде чем приступать к моделированию, нужно выполнить нормализацию и стандартизацию данных путём их масштабирования. Масштабирование выполняется из-за того, что разные величины выражены в разных единицах измерения. Эта процедура позволяет организовать «честную схватку» между признаками при определении их важности.
