# DataAn Enhanced — Автоматический статистический анализ

Инструмент для автоматического статистического анализа данных с интерактивным CLI и Jupyter интерфейсами.  
Загрузите `.xlsx` / `.csv` файл, выберите переменные — и получите полный HTML-отчёт с визуализациями, проверкой гипотез, регрессией, отбором признаков, PCA, кластеризацией и ML-бенчмарком.

## Быстрый старт

```bash
pip install -r requirements.txt
python main.py path/to/data.xlsx
```

Или откройте Jupyter ноутбук `DataAn_Enhanced.ipynb`.

## Интерфейсы

| Интерфейс        | Файл                    | Описание                                                                     |
|------------------|-------------------------|------------------------------------------------------------------------------|
| **CLI**          | `main.py`               | Интерактивный терминал. Поддерживает `main.py file.xlsx` для прямого запуска |
| **Jupyter**      | `DataAn_Enhanced.ipynb` | Виджеты для выбора параметров и генерации отчёта                             |
| **Библиотека**   | `analyzer_enhanced.py`  | Класс `DataAnalyzer` — импортируйте и используйте программно                 |

## Требования к данным

- Формат: `.xlsx` (первый лист) или `.csv`
- Минимум **1 категориальный** столбец (группирующая переменная)
- Минимум **1 числовой** столбец (целевая / многомерные признаки)

## Этапы анализа

### 0. Предобработка
- Удаление пропущенных значений
- Удаление выбросов по Z-оценке (настраивается, по умолчанию ±3σ)
- Фильтрация высококоррелированных признаков (|r| > 0.9, настраивается)
- Бутстреп-выравнивание групп (опционально)
- Сохранение сессии (`analyzer_session.json`) — восстанавливает параметры последнего файла

### 1. Визуализация (8 типов графиков)
- Скрипичная + роевая диаграммы со статистикой
- Ящик с усами (boxplot) с обозначением значимых различий (`*`, `**`, `***`); выбросы отключаются в конфиге
- Гистограммы со средним / медианой / модой
- Круговая диаграмма (пропорции категорий)
- Точечный график с линией регрессии
- PairGrid (верх — scatter, низ — KDE, диагональ — гистограмма)
- Корреляционная матрица (незначимые связи полупрозрачны)
- График взаимодействия (двухфакторный ANOVA)

### 2. ANOVA / Категориальные
- **Однофакторный ANOVA** (параметрический, проверка нормальности + гомогенности) или **Kruskal-Wallis** (непараметрический)
- **Пост-хок**: Tukey HSD после ANOVA; **тест Данна с поправкой Холма** после Kruskal-Wallis
- **Двухфакторный ANOVA** с проверкой взаимодействия; если взаимодействие значимо → анализ простых эффектов (t-тест Уэлча или однофакторный ANOVA внутри каждого уровня второго фактора с поправкой Бонферрони)
- Критерий χ² / точный тест Фишера + V Крамера для пар категориальных признаков
- Таблицы частот

### 3. MANOVA
- Все четыре критерия: Pillai, Wilks, Hotelling, Roy
- Пост-хок MANOVA (Tukey HSD для каждой зависимой переменной)

### 4. Регрессионный анализ
- **Линейная регрессия** — R², RMSE, MAE, коэффициенты, диагностические графики (остатки vs предсказанные, Q‑Q)
- **Логистическая регрессия** — категориальные + числовые предикторы → классификация групп, точность cross-val

### 5. Отбор признаков
- **Random Forest** — столбчатая диаграмма важности признаков с подписями значений
- **RFE** (рекурсивное устранение) с деревом решений

### 6. PCA
- График «каменистая осыпь» (индивидуальная + кумулятивная дисперсия, порог 95%)
- Тепловая карта нагрузок (первые 5 компонент)

### 7. Кластерный анализ
- **Метод локтя** — определение оптимального числа кластеров
- **K-средних** — кластеризация с проекцией на PCA, центроиды
- **ANOVA для кластеров** — однофакторный ANOVA каждого признака по кластерам
- Boxplot признаков по кластерам
- **Профили кластеров** — линейный график с заливкой ±1σ (среднекв. отклонение)
- Распределение категориальных признаков по кластерам
- Экспорт в XLSX с колонкой `cluster`

### 8. Машинное обучение (бенчмарк)
- Модели: Random Forest, Logistic Regression, Decision Tree, SVM (RBF), SVM (Poly), LDA, XGBoost (опционально)
- 10-кратное повторное деление на train/test
- Точность ± std, AUC, таблица рейтинга
- Матрица ошибок + ROC-кривая для лучшей модели

## Выходной отчёт

HTML-отчёт `{filename}_report.html` создаётся в рабочем каталоге:

- Оглавление с якорями
- Все графики встроены как base64 (клик для увеличения)
- Пояснения к каждому статистическому тесту
- Секции пользовательских комментариев
- Адаптивный CSS

## Конфигурация

Создайте `analyzer_config.json` в рабочем каталоге:

```json
{
  "precision": 3,
  "correlation_threshold": 0.9,
  "z_score_threshold": 3.0,
  "remove_outliers": true,
  "balance_groups": false,
  "bootstrap_min_size": 30,
  "bootstrap_max_ratio": 3.0,
  "ml_n_repeats": 10,
  "show_boxplot_outliers": true
}
```

## Сохранение сессии

Инструмент сохраняет путь к последнему файлу и выбранные параметры в `analyzer_session.json`.  
При повторной загрузке того же файла параметры восстанавливаются автоматически.
